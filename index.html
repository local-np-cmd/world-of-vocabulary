📚 Noun
नाम
Names a person, place, thing, or idea

✨ Examples:

teacher / शिक्षक Nepal / नेपाल book / पुस्तक happiness / खुशी
🔁 Pronoun
सर्वनाम
Replaces a noun

✨ Examples:

I / म you / तिमी he / ऊ she / उनी it / यो we / हामी they / उनीहरू
⚡ Verb
क्रिया
Action or state of being

✨ Examples:

run / दौडनु eat / खानु sing / गाउनु is / छ become / हुनु
🎨 Adjective
विशेषण
Describes a noun or pronoun

✨ Examples:

big / ठूलो beautiful / सुन्दर red / रातो happy / खुशी tall / अग्लो
🏃 Adverb
क्रियाविशेषण
Describes verb, adjective, or another adverb

✨ Examples:

quickly / छिटो slowly / ढिलो very / धेरै well / राम्रोसँग yesterday / हिजो
📍 Preposition
पूर्वसर्ग
Shows position, time, or direction

✨ Examples:

in / भित्र on / माथि under / मुनि beside / छेउमा before / अगाडि
🔗 Conjunction
संयोजक
Connects words, phrases, or sentences

✨ Examples:

and / र but / तर or / वा because / किनभने although / यद्यपि
❗ Interjection
विस्मयादिबोधक
Expresses sudden feelings or emotions

✨ Examples:

Wow! / वाह! Oh! / ओहो! Ouch! / आहा! Hooray! / हुर्रे!
🔍 Determiner/Article
निर्धारक
Introduces or identifies a noun

✨ Examples:

a / एक an / एक the / त्यो this / यो that / त्यो some / केही
